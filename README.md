# DO!CLI

A modern TUI todo manager with a cappuccin tokyo night theme and ASCII banner.

## Features
- Keyboard-first TUI
- Cappuccin Tokyo Night palette
- JSON-backed todo storage

## Run

```bash
go mod tidy
go run .
```

## Keys
- `a` add a new task
- `c` toggle complete
- `d` delete task
- `q` quit
